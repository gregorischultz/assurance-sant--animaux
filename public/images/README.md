# Images à fournir

Placez ici les visuels du site (optimisés WebP/AVIF de préférence) :

| Fichier | Usage | Dimensions conseillées |
|---|---|---|
| `logo.png` | Logo pour JSON-LD Organization | 512×512 |
| `og-default.png` | Open Graph / Twitter par défaut | 1200×630 |
| `favicon.ico` | Favicon (à placer dans `/public`) | 32×32 |
| `guides/*.jpg` | Couvertures d'articles (frontmatter `cover`) | 1200×675 (16:9) |
| `authors/*.jpg` | Photos des auteurs | 200×200 |

Tant qu'elles ne sont pas fournies, les composants affichent des placeholders
gris (pas d'image cassée). Les métadonnées OG pointent vers `og-default.png`.
